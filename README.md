# Speats Website

Static HTML/CSS/JavaScript landing page presenting Speats and its AI products.

## About

Speats is a Swiss AI solutions company. This site showcases the company's services and products:

- **Odoo AI** — AI-powered product description generator for Odoo
- **FPH Prep** — Exam preparation platform for Swiss pharmacists (FPH Offizin)
- **BrokerAI** — AI platform for insurance policy analysis and management

## Tech Stack

- Static HTML, CSS, vanilla JavaScript — no build system
- Multi-language support (EN, IT, FR, DE) via a JavaScript i18n system
- Mobile-first responsive design
- SEO optimized (structured data, Open Graph, sitemap)
